option_list() → Sequence[Union[str, Tuple[str, str]]][source]¶

Implement this.

This method returns the list of options. An option is either a button label, or a tuple formed by a button label and a button code. The code is returned through the inline query mechanism of telegram if available, otherwise, the label is.

See telegram.inlinequery.

docker_utils.emoji_of_status(status: str) → str[source]¶

Returns the emoji associated to a docker container status.

The emojis are as follows:

  • exited: ⏹,

  • paused: ⏸,

  • restarting: ↩,

  • running: ▶,

  • otherwise: ❓.
